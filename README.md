# 🤖 AI Chatbot Project

An intelligent **AI-powered chatbot application** built using **Spring Boot (Backend)** and **React (Frontend)**.
The chatbot integrates with **Google Gemini AI API** to generate real-time responses.

---

## 🚀 Features

* 🤖 AI-based chatbot responses
* ⚡ Fast REST API using Spring Boot
* 💬 Real-time chat interface with React
* 🧠 Gemini AI API integration
* 🗄️ MySQL database integration
* 🔒 Clean backend architecture

---

## 🏗️ Project Architecture

Frontend and Backend are maintained in a **single repository**.

```
ai-chatbot-project
│
├── Backend
│   ├── src/main/java
│   ├── src/main/resources
│   │   └── application.properties
│   └── pom.xml
│
├── Frontend
│   ├── src
│   ├── public
│   └── package.json
│
└── README.md
```
